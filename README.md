# MPFastDevLibrary
我的快速开发平台。my fast development platform。

## 我的工具帮助类

### SerializableHelper
序列化辅助类
二进制和xml的序列化和反序列化

### MapHelper
映射工具类
类映射，用于同属性名类快速拷贝

### EnumHelper
枚举帮助类
用于枚举的相关操作，获取枚举项集合，获取枚举描述和描述集合

### CsvHelper
Csv读写帮助类
支持直接读取行，读写DataTable等
